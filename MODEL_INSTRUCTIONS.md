Testsat is a modern test taking application built using mern stack.It is purely javascript based application. it uses clerk for authentication and cloudinary for file upload. it has admin and student roles. admin can add tests, view all tests, edit tests, delete tests, view all submissions, view all students, and manage students. student can take tests, view their test history, and view their test results. Testsat supports latex so admin can add latex equations in the test questions, answers, and explanations. it also has a test taking guide for students to help them prepare for the test. it also has a submission history for students to view their test history. it also has a test results page for students to view their test results. it also has a logout button for students to logout of their account. When students take test there is a timer and when the timer is up the test is submitted automatically. Test questions are structured as testname > sections > modules> questions> mcq and images. mcq has options and answer. answer should be a dropdown of options from mcq options.
